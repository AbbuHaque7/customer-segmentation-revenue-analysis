# customer-segmentation-revenue-analysis
This project analyzes customer segmentation and revenue patterns to identify high-value customer groups and potential business opportunities.
The analysis was performed using Excel for data preparation and Power BI for interactive dashboard visualization.

# Project Objective
The goal of this project is to understand how different customer segments contribute to revenue and to identify strategic opportunities for expanding the customer base and increasing revenue generation.

# Dataset
The dataset used in this project is a synthetic dataset generated for educational and portfolio purposes.

Main features include:
  * Customer ID
  * Age & Age Group
  * Gender
  * City
  * Annual Income
  * Spend Score
  * Spend Behaviour
  * Total Orders
  * Average Order Value
  * Last Purchase Days Ago
  * Loyalty Status
  * Preferred Channel
  * Profile Richness

# Data Preparation
 Data preparation was performed in Excel, including:
 * Data cleaning
 * Age group categorization
 * Spend behaviour classification
 * Profile richness flag creation
 * Segment labeling for customer analysis
The cleaned dataset was then used for visualization in Power BI.

# Dashboard Overview

 The Power BI dashboard explores multiple customer segmentation perspectives.
  * Revenue Overview
    Shows how revenue is distributed across different income and age groups.
  * Cross Segmentation
    Analyzes the distribution of customer spending behavior across income segments.
  * Recency Segmentation
    Identifies how recently customers have made purchases.
  * Frequency Segmentation
    Shows how frequently different customer segments make purchases.
  * Monetary Segmentation
    Analyzes customer contribution based on total spending.
  * Strategic Opportunities
    Highlights potential expansion opportunities across income and spending segments.
