# customer-segmentation-revenue-analysis
This project analyzes customer segmentation and revenue patterns to identify high-value customer groups and potential business opportunities.
The analysis was performed using Excel for data preparation and Power BI for interactive dashboard visualization.

# Project Objective
The goal of this project is to understand how different customer segments contribute to revenue and to identify strategic opportunities for expanding the customer base and increasing revenue generation.

# Dataset
The dataset used in this project is a synthetic dataset generated for educational and portfolio purposes.

Main features include:
  * Customer ID
  * Age & Age Group
  * Gender
  * City
  * Annual Income
  * Spend Score
  * Spend Behaviour
  * Total Orders
  * Average Order Value
  * Last Purchase Days Ago
  * Loyalty Status
  * Preferred Channel
  * Profile Richness
